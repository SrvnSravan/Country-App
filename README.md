# 🌍 Country App

A responsive React web application that displays countries filtered by region. It features a dynamic image carousel, a static highlight image, and a sticky header with a responsive collapsible menu for mobile users.

---

## 🚀 Live Demo

👉 [View Live on Vercel](https://country-app-1tri.vercel.app)

---

## ✨ Features

- 🔍 **Filter by Region** — View countries from "All", "Asia", or "Europe"
- 🖼️ **Image Carousel** — Auto-rotating slider showcasing scenic backgrounds
- 📱 **Mobile Friendly** — Hamburger menu for smooth mobile experience
- 📌 **Sticky Header** — Always-visible navigation for ease of access

---

## ⚙️ Tech Stack

- **React** with Functional Components and Hooks
- **Redux Toolkit** for state management
- **React Bootstrap** for layout and carousel
- **Bootstrap 5** for responsive styling
- **Vercel** or **Netlify** for deployment

---

## 🛠️ Getting Started

To run the app locally:

```bash
git clone https://github.com/SrvnSravan/Country-App.git
cd country-app
npm install
npm start
```

---

## 📦 Build for Production

To create a production build:

```bash
npm run build
```

This will generate a `/build` folder that can be deployed to:

- [Netlify](https://www.netlify.com/)
- [Vercel](https://vercel.com/)
- [GitHub Pages](https://pages.github.com/)
